# 🚀 Fine-Tuning DeBERTa-v3 using Google Colab (Free GPU)

This guide provides the exact code needed to train your local routing model absolutely free in ~15 minutes.

## Step 1: Open Google Colab
1. Go to [colab.research.google.com](https://colab.research.google.com/)
2. Click **New Notebook**
3. Most Importantly: Go to `Runtime` -> `Change runtime type` -> Select `T4 GPU` (This is FREE).

## Step 2: Upload Your Dataset
1. On the left sidebar of Colab, click the **Folder icon** (Files).
2. Upload the `training_data.csv` file that was generated by your `scripts/harvest_training_data.py`.

## Step 3: Paste and Run This Code
Create a code block in your Colab notebook, paste the following, and click Run.

```python
# 1. THE NUCLEAR OPTION: Uninstall and reinstall pinned stable versions
# This resolves both the 'default_logdir' (transformers) and 'DatasetFilter' (huggingface_hub) errors.
!pip uninstall -y setfit transformers huggingface_hub datasets tokenizers
!pip install -q "setfit==1.0.3" "transformers<4.40.0" "huggingface_hub<0.24.0" "datasets<2.20.0"

# 2. FORCE RESTART SESSION (Critical)
import os
os.kill(os.getpid(), 9)

import pandas as pd
from datasets import Dataset
from setfit import SetFitModel, Trainer, TrainingArguments
import torch

print(f"GPU Available: {torch.cuda.is_available()}")

# 2. Load the uploaded dataset
print("Loading data...")
df = pd.read_csv('training_data.csv')

# Check distribution
print("Category Distribution:")
print(df['label'].value_counts())

# Convert to HuggingFace Dataset
dataset = Dataset.from_pandas(df)

# Split into train/test (80/20)
dataset = dataset.train_test_split(test_size=0.2, seed=42)
train_dataset = dataset["train"]
eval_dataset = dataset["test"]

# 3. Download the pre-trained DeBERTa-v3-small
print("Downloading base model...")
model = SetFitModel.from_pretrained(
    "microsoft/deberta-v3-small",
    labels=df['label'].unique().tolist()
)

# 4. Set up the Trainer
args = TrainingArguments(
    batch_size=16,
    num_epochs=3,
    evaluation_strategy="epoch",
    save_strategy="epoch",
    load_best_model_at_end=True,
    run_name="ai-router-deberta"
)

trainer = Trainer(
    model=model,
    args=args,
    train_dataset=train_dataset,
    eval_dataset=eval_dataset,
)

# 5. Train!
print("Starting fine-tuning...")
trainer.train()

# 6. Evaluate accuracy
metrics = trainer.evaluate()
print(f"Final Accuracy: {metrics}")

# 7. Save and ZIP the model for download
print("Saving model natively...")
model.save_pretrained("./deberta-router")

# Zip the folder to make it easy to download
!zip -r deberta-router.zip ./deberta-router

print("✅ DONE! You can now download deberta-router.zip from the files sidebar.")
```

## Step 4: Download and Add to Project
1. Once training finishes, you will see `deberta-router.zip` in your Colab files perfectly formatted.
2. Download it, unzip it, and place the `deberta-router` folder inside a new `d:\final router\models\` directory.
3. Your local router is ready!
